## **Parking Reservation System**

The **Parking Reservation System** is a Java application that helps users find and reserve parking spots in advance. It features a dark-themed GUI built with **Java Swing** and uses **MySQL** for secure data storage.  

Users can register, log in, select a city, book a spot, make payments, and receive a **QR code** for verification. The system aims to reduce traffic congestion and unorganized parking through a simple, efficient interface.

---

### **Technologies Used**
- Java (Core + OOP)  
- Java Swing  
- MySQL  
- JDBC  
- QR Code Generator  

---

### **Setup**
1. Create a MySQL database named `parking_db`.  
2. Update your database credentials in the Java code.  
3. Open the project in NetBeans, Eclipse, or VS Code.  
4. Run `Login.java` to start the app.
